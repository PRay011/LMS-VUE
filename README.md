# LiblalyManane-Vue
注意事项：
由于各个电脑的高度宽度不同，最大程度避免使用绝对数据来定位
尽量使用百分比数据
例如：
margin-left: 100px （打咩）
margin-left: 10%    (bingo)
拉取后需要在项目中修改相应的位置高度：
login.css ：5、6行
index.css ：3、4行
index.css : 67行
index.vue ：6行
library.css : 3、4行
selfCenter.css : 2、3、82行
selfCenter.vue: 21、53行....(还有几个类似的自己看着办吧qwq)
detail.vue:39行
